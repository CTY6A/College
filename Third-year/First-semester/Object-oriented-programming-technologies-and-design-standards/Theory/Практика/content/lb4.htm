Лабораторная работа №4

Моделирование взаимодействия объектов системы при помощи диаграмм sEQUENCE и collaboration

Цель работы:

·     научиться строить диаграммы  Sequence и Collaboration в объектно-ориентированной среде автоматизированного синтеза;

·     разработать диаграммы Sequence  и Collaboration для проектируемой системы.

 

Задание:

1. С помощью диаграммы  Sequence получить отражение во времени процесса обмена сообщениями между объектами создаваемой системы.

2.  С помощью диаграммы Collaboration получить отражение взаимодействия всех объектов создаваемой системы в конкретный момент времени.

 

Порядок выполнения работы

 

1. Изучить теоретическую часть лабораторной работы.

2. Ответить на контрольные вопросы.

3. Разработать диаграмму Sequence по предложенной тематике.

4. Разработать диаграмму Collaboration по предложенной тематике.

 

 

Контрольные вопросы

1.  Какие различия между диаграммами Sequence и Collaboration и что отображается на каждой из диаграмм?

2.     Какие виды сообщений позволяет отразить диаграмма Sequence?

3.  Для отображения обмена сообщениями между объектами используются следующие виды диаграммы Sequence:

        а) Sequence: Instance;  б) Sequence: Role;  в) Sequence: Object.

4. Сообщение Message от одного объекта к другому может обозначать:

а) вызов операций класса; б) передачу управления вызываемому объекту; в) передачу управления самому себе; г) все ответы правильные.

5.  Для отображения времени жизни объекта необходимо использовать следующие элементы:

        а) Lifeline Actor;  б) Lifeline;  в) оба варианта правильные.

6.  Как перенести данные между диаграммами Sequence и Collaboration  и всегда ли корректен этот перенос?
