МЕТОДИЧЕСКИЕ УКАЗАНИЯ К ВЫПОЛНЕНИЮ ЛАБОРАТОРНЫХ РАБОТ

по дисциплине «Языки программирования. Часть 2»

 

Перечень лабораторных работ

Программой дисциплины «Языки программирования» по второй части предусмотрено выполнение восьми лабораторных работ на алгоритмическом языке Си по следующим темам:

1.     Основные элементы языка.

2.     Операторы управления.

3.     Циклы.

4.     Функции.

5.     Динамические структуры.

6.     Сложные динамические структуры.

7.     Файлы.

8.     Макросредства.

 

Перечень компьютерных программ, необходимых для выполнения лабораторных работ

 

1.       Операционная система Windows.

2.       Среда программирования (Microsoft Visual Studio .NET).

3.       Среда разработки Borland C++ 3.1.

 

Указания к выполнению лабораторных работ

 

Целью каждой из лабораторных работ является разработка и отладка программы на языке Си, соответствующей теме работы и индивидуальному заданию.

Индивидуальные задания по каждой лабораторной работе выдает преподаватель.

Краткие пояснения к средам выполнения лабораторных работ представлены в описании Среды разработки, приведенном в разделе «Практика» ЭРУД.

Для успешного выполнения лабораторной работы необходимо предварительно изучить соответствующие темы из раздела «Теория» ЭРУД, ответить на вопросы тестов по соответствующим темам (Тесты к теоретическому разделу приведены в разделе «Контроль знаний» ЭРУД, правильные ответы по каждому из контрольных вопросов тестов приведены после всех тестов к теоретическому разделу).

Помощь в выполнении лабораторных работ по второй части дисциплины призвано оказать Учебно-методическое пособие «Конструирование программ и языки программирования. Язык Си» [1], приведенное в разделе «Практика» ЭРУД. По каждой из соответствующих лабораторных работ в пособии приведены краткие теоретические сведения и индивидуальные задания.

 

Первая лабораторная работа посвящена теме «Основные элементы языка». Перед ее выполнением необходимо изучить темы 1 – 2 («Основные типы данных  23», «Операции и выражения  38  48») из теоретического материала ЭРУД и раздел «Общие теоретические положения» из материалов учебно-методического пособия, а также ответить на вопросы тестов по темам 1, 2 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

Вторая лабораторная работа посвящена теме «Операторы управления». Перед ее выполнением необходимо изучить тему 3 («Операторы управления  71») из теоретического материала ЭРУД и теоретические сведения из лабораторной работы №1 («Разработка программ с использованием условных  и циклических операторов») учебно-методического пособия, а также ответить на вопросы тестов по теме 3 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

Третья лабораторная работа посвящена теме «Циклы». Для ее выполнения необходимо знать темы 3, 4, 6 («Операторы управления», «Массивы и указатели», «Строки  87») из теоретического материала ЭРУД и теоретические сведения из лабораторных работ №2 – 4 («Массивы в языке Си», «Работа со строками», «Работа со структурами») учебно-методического пособия, а также ответить на вопросы тестов по темам 3, 4, 6 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

Четвертая лабораторная работа посвящена теме «Функции». Перед ее выполнением необходимо изучить тему 5 («Функции  99») из теоретического материала ЭРУД, а также ответить на вопросы тестов по теме 5 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

Пятая и шестая лабораторные работы посвящены темам «Динамические структуры» и «Сложные динамические структуры» соответственно. Для их выполнения необходимо изучить темы 8, 9 («Структуры, объединения и перечисления», «Динамические структуры данных  124») из теоретического материала ЭРУД и теоретические сведения из лабораторной работы №6 («Динамические структуры данных») учебно-методического пособия, а также ответить на вопросы тестов по темам 8, 9 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

Седьмая лабораторная работа посвящена теме «Файлы». Перед ее выполнением необходимо изучить тему 10 «Файлы» из теоретического материала ЭРУД и теоретические сведения из лабораторной работы №5 («Работа с файлами») учебно-методического пособия, а также ответить на вопросы тестов по теме 10 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

Восьмая лабораторная работа посвящена теме «Макросредства». Перед ее выполнением необходимо изучить тему 11 «Директивы препроцессора  183» из теоретического материала ЭРУД, а также ответить на вопросы тестов по теме 11 из «Тестов к теоретическому разделу» раздела «Контроль знаний» ЭРУД.

 

При сдаче лабораторной работы студент должен ответить на теоретические вопросы преподавателя по соответствующей теме, а также на вопросы, касающиеся программы, разработанной в соответствии с индивидуальным заданием. Каждая лабораторная работа студента оценивается оценкой. По совокупности оценок по всем лабораторным работам студент получает итоговую оценку. Данная оценка учитывается при сдаче студентом теоретического экзамена по второй части дисциплины.
